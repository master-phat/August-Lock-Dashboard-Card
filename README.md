# August-Lock-Dashboard-Card
Premium August Lock-inspired smart lock card for Home Assistant Button Card. Features animated LED chase ring, metallic floating lock puck, SVG logo support, panels, battery status, door state, recent activity, and customizable entities.

### Installation

1. Install Button Card through HACS

2. Upload `august-logo.svg` to: /config/www/icons/ using: Studio Code Server

3. Open your dashboard

4. Click: Edit Dashboard → Add Card

5. Choose: Manual Card

6. Open: Show Code Editor

7. Paste the provided YAML

8. Verify your entities match:

lock.front_door  
sensor.front_door_battery  
binary_sensor.front_door_door

9. Save & Enjoy your Animated Premium August lock UI 😆
